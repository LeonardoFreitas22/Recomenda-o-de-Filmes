# Recomenda-o-de-Filmes
Desenvolvi um sistema de recomendação de filmes que utiliza Machine Learning e análise de texto para sugerir conteúdos com base na similaridade entre sinopses. A solução melhora a experiência do usuário ao indicar títulos relevantes, reduzindo o tempo de busca e podendo ser aplicada em streaming, e-commerce e outras plataformas digitais.
